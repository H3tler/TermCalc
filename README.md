# Usage

### Operators in descending order of their precedence:
1. **'^'**
2. **'\*', '/', '%'**
3. **'+', '-'**

* **-Use "ans" to reference the result of the previous expression.**

**Provide the 'test' flag if you want to run tests**

    ./TermCalc test
